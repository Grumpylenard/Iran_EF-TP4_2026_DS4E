# Poster Graphics Folder

This folder contains the 5 selected graphics for the poster:

## Expected Files:
- map-1.pdf (or similar) - Large map 1
- map-2.pdf (or similar) - Large map 2  
- map-3.pdf (or similar) - Large map 3
- graph-1.pdf (or similar) - Small graphic 1
- graph-2.pdf (or similar) - Small graphic 2

## Usage:
Drag and drop your 5 selected PDF/PNG files here, then the poster will reference them directly.

## File Requirements:
- High resolution (300+ DPI)
- PDF format preferred
- Consistent naming for easy reference
